# NexEvent - Conference & Hackathon Finder

A premium web application to discover Conferences and Hackathons in India and Virtual spaces.

## Features
- **Discover**: Browse curated lists of events.
- **Filter**: 
  - **Location**: India (Physical) or Virtual.
  - **Type**: Hackathons or Conferences.
  - **Indexing**: IEEE, Scopus, Web of Science (WoS).
- **Details**: View pricing (APC/Registration), submission dates, and event dates.

## How to Run
1. Simply double-click `index.html` to open it in your browser.
2. No installation required.

## Tech Stack
- HTML5
- CSS3 (Vanilla, Glassmorphism design)
- JavaScript (Vanilla)
